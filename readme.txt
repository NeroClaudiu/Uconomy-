How to install:
Go to your Unturned root server directory and go to Servers/Your_Server_Name/Rocket and extract the Libraries and the Plugins folders! 
Don't forget to configure your MySQL server(first you need to open the server then stop it, go to the plugins folder, open uConomy folder and edit Uconomy.configuration.xml).
